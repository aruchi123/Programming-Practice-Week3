# Programming-Practice-Week3
Week 3 Programming Practice Module containing Merge Sort, Quick Sort and Heap Sort Implemented in java.
This repository contains solutions for Week 3 Programming Practice (3rd Year), including Merge Sort, Quick Sort and Heap Sort with output Screenshots.
# Merge Sort:
1: The final sorted list will help the university identify minimum, median, and maximum packages while generating analytical reports. 
2: The solution should efficiently handle large datasets and demonstrate a clear understanding of Merge Sort's divide-and-conquer strategy. 
# Quick Sort:
1: task is to implement Quick Sort to sort the response times in ascending order. Built-in sorting methods are not allowed. 
2: task is to implement Quick Sort to arrange the trade values from highest to lowest. 
# Heap Sort:
1: task is to implement Heap Sort and arrange all player scores in increasing order. 
2: Hospital administrators want to analyze the efficiency of their emergency services. To do so, all response times must first be sorted using Heap Sort.
